Improvement example golang app structure for web project

Origin post https://dwahyudi.github.io/2020/07/26/golang-crud-gin-html.html and repo:

```bash
github.com/dwahyudi/golang_grocery_sample
```

Prerequisits:
1. Linux + Mysql + Golang 1.1X.X
2. Free port on localhost:8080

Run webapp:

1. Make new database "grocery" with user "grocery" and password "pass_to_grocery"
2. Import db_init.sql
3. Run in terminal:

```bash
chmod +x run.sh
./run.sh
```
